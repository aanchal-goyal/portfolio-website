# 💼 Personal Portfolio Website

A modern **developer portfolio website** built using **HTML, CSS, and JavaScript**.  
This website showcases my projects, skills, education, and contact information.

It is designed to provide a simple and professional way for recruiters and collaborators to explore my work.
